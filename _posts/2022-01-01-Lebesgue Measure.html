Let $G$ be an open subset of $ℝ$. For $x, y∈G$, let $I_{x, y}$ be the closed (or open, if you prefer) interval between $x$ and $y$, so $I_{x, x}=\{x\}$ (or ∅). Define a relation ∼ on $G$ by $x∼y$ if and only if $I_{x, y}⊂G$.
  1. Show that ∼ is an equivalence relation on $G$.
  2. Show that each equivalence class is an open interval.
  3. Show that there are (at most) countably many equivalence classes.
  4. Deduce that $G$ is the union of (at most) countably many, disjoint open intervals.
  1. If $x∈G$, then $I_{x,x}=\{x\}⊂G$, so $x∼x$
    If $x∼y$, then $I_{y,x}=I_{x,y}⊂G$, so $y∼x$
    If $x∼y,y∼z$, then $I_{x,z}⊂I_{x,y}∪I_{y,z}⊂G$, so $x∼z$
  2. Let $A$ be an equivalence class. By metric space Theorem 7.2.1. $A$ is an interval.
    For any $a∈A$, since $G$ is open, $∃r>0:B(a,r)⊂G$. Note that $∀b∈B(a,r):b∼A$. So $B(a,r)⊂A$. So $A$ is open.
  3. Each rational number is in one equivalence class and each equivalence class contains some rational numbers. We constructed a surjection from ℚ to $G/$∼ and $G/$∼ is infinite, so $G/$∼ is countable.
  4. $G$ is the union of equivalence classes, which are disjoint open intervals. We proved they are countable.
