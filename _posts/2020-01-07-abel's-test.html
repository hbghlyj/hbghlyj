This note is an exposition of Abel’s test on convergence of series.

Theorem 1. Suppose ∑ 1 ∞ b n converges and that {an} is a monotone bounded sequence. Then ∑ 1 ∞ a n b n converges.

Proof. Let b0 = 0, B N = ∑ k = 0 N b k . Then bn = Bn − Bn − 1, n ≥ 1, hence ∑ k = 1 N a k b k = ∑ k = 1 N a k ( B k − B k − 1 ) = B 1 ( a 1 − a 2 ) + B 2 ( a 2 − a 3 ) + … B N − 1 ( a N − 1 − a N ) + a N B N = ∑ k = 1 N − 1 B k ( a k − a k + 1 ) + a N B N Since {an} is monotone and bounded it converges; and {BN} converges since ∑bn converges. Hence aNBN converges. We estimate ∑Bk(ak−ak+1). Since ∑bn converges, |∑bn| ≤ M for some M. Using the fact that {an} is montone we get ∑ 1 N | B k ( a k − a k + 1 ) | ≤ M ∑ 1 N | a k − a k + 1 | = M | a 1 − a N + 1 | → M | a 1 − a | , where ak → a. Hence ∑Bk(ak − ak+1) converges absolutely.

Example 1. Suppose ∑an converges. Then ∑n1/nan converges and ∑(1+1/n)nan converges.

Proof. It’s easy to prove that f(x) = x1/x is decreasing for x > e, by computing the derivative of (log(x))/x. Here is the proof that (1+1/n)n increases with n.

( 1 + 1 n ) n = 1 + 1 + ⋯ + ( n p ) 1 n p + ⋯ + 1 n n = 1 + ⋯ + ( 1 − 1 n ) … ( 1 − p − 1 n ) 1 p ! + ⋯ + 1 n n ( 1 + 1 n + 1 ) n + 1 = 1 + ⋯ + ( 1 − 1 n + 1 ) … ( 1 − p − 1 n + 1 ) 1 p ! + ⋯ + 1 ( n + 1 ) n + 1 . The last sum has one more (positive) term than the preceding term and the pth term of the last sum is larger than the preceding pth term since each factor is larger (subtract k/(n+1) from 1 as opposed to subtracting k/n). Hence ( 1 + 1 n + 1 ) n + 1 > ( 1 + 1 n ) n . It’s easy check using L’Hopital’s rule that (1+1/x)x → e as x → ∞, so the sequence (1+1/n)n is monotone and bounded. {% include qed.html %}
