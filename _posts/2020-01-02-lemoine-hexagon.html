--- tags: euclidean-geometry --- 过三角形的类似重心作三边的平行线,与各边的六个交点形成的六边形称为Lemoine六边形,它内接于第一Lemoine圆.

Lemoine六边形有两种画法,一种是简单六边形,顶点是前述的六个交点;第二种是自交六边形,以三条平行于三角形的边的直线作为边.
设三角形的边长为a,b,c,面积为 Δ ,
简单六边形的周长为 p = a 3 + b 3 + c 3 + 3 a b c a 2 + b 2 + c 2 ,面积为 K = a 4 + b 4 + c 4 + a 2 b 2 + b 2 c 2 + c 2 a 2 ( a 2 + b 2 + c 2 ) 2 Δ
自交六边形的周长为 p = ( a + b + c ) ( a b + b c + c a ) a 2 + b 2 + c 2 ,面积为 K = a 2 b 2 + b 2 c 2 + c 2 a 2 ( a 2 + b 2 + c 2 ) 2 Δ
