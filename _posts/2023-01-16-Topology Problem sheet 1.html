Let $𝒯_\text{left}$ be the family of subsets $U$ of ℝ with the property that $∀x∈U∃ε>0$ such that $(x-ε,x]⊂U$.
Prove that $𝒯_\text{left}$ is a topology on ℝ.
What is the closure of $(0,1)$ with respect to this topology?
(T1) is trivial.
(T2) Let $U,V∈𝒯_\text{left}$. Consider an arbitrary $x∈U∩V$.
As $U∈𝒯_\text{left},∃α_x>0$ such that $(x-α_x,x]⊂U$. As $V∈𝒯_\text{left},∃β_x>0$ such that $(x-β_x,x]⊂V$.
Then $(x-δ_x,x]⊂U∩V$ for $δ_x=\min(α_x,β_x)$. So $U∩V∈𝒯_\text{left}$.
(T3) Let $U_i∈𝒯_\text{left}$. Consider an arbitrary $x∈⋃_{i∈I}U_i,∃k∈I:x∈U_k$.
As $U_k∈𝒯_\text{left}$, $∃ε_x>0:(x-ε_x,x]⊆U_k⊆⋃_{i∈I}U_i$. So $⋃_{i∈I}U_i∈𝒯_\text{left}$.
Therefore $𝒯_\text{left}$ is a topology on ℝ.
If $1∈ℝ∖\overline{(0,1)}$, then $∃ε>0:(1-ε,1]⊂ℝ∖\overline{(0,1)}$, but $1-\fracε2∈(1-ε,1]∩(0,1)$, contradiction.
So $1∈\overline{(0,1)}⇒(0,1]⊂\overline{(0,1)}$.
Also $ℝ∖(0,1]∈𝒯_\text{left}$, so $\overline{(0,1)}⊂(0,1]$. Therefore $\overline{(0,1)}=(0,1]$.

Let $(X,𝒯)$ be a Hausdorff topological space and let $f: X→X$ be continuous.
Show that the set$\DeclareMathOperator{\fix}{Fix}$$$\fix(f)=\{x:f(x)=x\}$$is closed.
$F:X→X×X;x↦(x,f(x))$ is continuous.
Since $X$ is Hausdorff, $\Delta=\{(x,x):x∈X\}$ is a closed set in $X×X$.
So $\fix(f)=F^{-1}(\Delta)$ is closed.
Alternate method:
To prove $X∖\fix(f)$ is open, for any $x∉\fix(f)$ we need to find open $U∋x$ such that $U∩\fix(f)=∅$.
By Hausdorff, we can find disjoint open sets $U_x∋x$ and $U_{f(x)}∋f(x)$. By continuity, $f^{-1}(U_{f(x)})$ is open, so $U_x∩f^{-1}(U_{f(x)})$ is open. Clearly $x∈U_x∩f^{-1}(U_{f(x)})$. It remains to prove $\fix(f)∩U_x∩f^{-1}(U_{f(x)})=∅$.
Let $y∈\fix(f)∩U_x∩f^{-1}(U_{f(x)})$, then $y∈U_x$ and $y=f(y)∈U_{f(x)}$, contradicting $U_x,U_{f(x)}$ are disjoint.
