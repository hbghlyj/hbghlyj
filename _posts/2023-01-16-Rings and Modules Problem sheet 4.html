IV.1. Let$$A=\pmatrix{3&0&0\\0&2+2\sqrt{-5}&0\\0&0&4}$$Explain why A is in Smith normal form over ℂ, but is not in Smith normal form over $ℤ[\sqrt{-5}]$. Is A equivalent to a matrix in Smith normal form over $ℤ[\sqrt{-5}]$?

A is diagonal. Since ℂ is a field, $a|b∀a,b∈ℂ^*$, so A is in Smith normal form.
A is not in Smith normal form over $ℤ[\sqrt{-5}]$, since $3∤2+2\sqrt{-5}$.
$\gcd(3,4)=1,\lcm(3,4)=12⇒A$ is equivalent to $$\pmatrix{1&0&0\\0&2+2\sqrt{-5}&0\\0&0&12}$$This is in Smith normal form, since $12=(2+2\sqrt{-5})(1-\sqrt{-5})$.

IV.3. Show that if A, B ∈ Mn(𝔽) are such that 𝔽n/𝔽nA is 𝔽-linearly isomorphic to 𝔽n/𝔽nB, then A and B are equivalent as matrices.

𝔽 is a field, so an 𝔽-module is a vector space,
dim 𝔽n/𝔽nA = dim 𝔽n/𝔽nB ⇒ n − dim 𝔽nA = n − dim 𝔽nB ⇒ dim 𝔽nA = dim 𝔽nB
⇒ 𝔽nA, 𝔽nB are isomorphic ⇒ A is equivalent to B.

IV.5. Determine the rational canonical form of the matrix$$A=\pmatrix{1&1&0&0\\0&1&0&0\\2&3&-1&4\\1&1&-1&3}$$

To transform a matrix on $ℚ[X]$ to SNF, we multiply it on both sides by invertible matrices.
  • The inverse operation of adding $f(x)$ times row $i$ to row $j$ is adding $-f(x)$ times row $i$ to row $j$, so $f(x)$ is allowed to be any element
  • The inverse operation of rescaling $i$th row/column by $f(x)$ is rescaling it by $f(x)^{-1}$, so $f(x)$ is only allowed to be units [$\deg f(x)>0$ disallowed]
Perform the SNF algorithm on $xI-A$

IV.6. Suppose that G is a finite commutative group. Show that there is an element x ∈ G such that the order of every y ∈ G divides the order of x. Assuming that every polynomial in 𝔽[X] of degree N has at most N roots show that U(𝔽) is a cyclic group when 𝔽 is a finite field.

By Corollary 7.2 G is isomorphic to $ℤ/⟨d_1⟩⊕⋯⊕ℤ/⟨d_n⟩$ for natural numbers $d_1∣⋯∣d_n$. Let x be the element of G corresponding to $(0,…,0,1)$ then x has the maximum order $d_n$, and the order of any element of G divides $d_n$.
Let |U(𝔽)| = n and x is an element of maximum order d, by Lagrange's theorem $d|n$.
We proved all elements of U(𝔽) have order dividing d, so they are roots of the polynomial $X^d-1$, which have$≤d$ roots in U(𝔽), so $n≤d$, so $n=d$, so U(𝔽) is generated by $x$.
Finite subgroups of the multiplicative group of a field are cyclic

IV.7. Suppose that R is a commutative ring and for a1, …, an ∈ Rncolwrite [a1, …, an] for the matrix in Mn(R) with columns a1, …, an in order. Show that the maps Rcoln→ R; ai ↦ det[a1, …, an] are R-linear for each 1 ≤ i ≤ n; and det[a1, …, an] = 0 whenever ai = aj for some i ≠ j. By considering det[Ba1, …, Ban] for B ∈ Mn(R) show that det BA = det B det A.

$$r\det[a_1,…,a_i,…,a_n]+s\det[a_1,…,a_i',…,a_n]=\det[a_1,…,ra_i+sa_i',…,a_n]$$So the maps ai ↦ det[a1, …, an] are R-linear.
If a = ai = aj for some i ≠ j. Since swapping columns negates the determinant
det[…, a, …, a, …] = −det[…, a, …, a, …]
So it is zero.
Write $a_i$ as linear combination of basis $e_i$:$$a_i=\sum_{j=1}^na_{j,i}e_j$$In the linear expansion, using $\det[…,e_j,…,e_j,…]=0$, we get \[\left.\begin{aligned}\det[Ba_1,…,Ba_n]&=\det[Ba_{11}e_1,Ba_{22}e_2,…,Ba_{nn}e_n]\\&+\det[Ba_{21}e_2,Ba_{12}e_1,…,Ba_{nn}e_n]\\&+⋯\\&+\det[Ba_{n1}e_n,Ba_{n-1,2}e_{n-1},…,Ba_{1n}e_1]\end{aligned}\right\}n!\text{ terms}\]Let $B=[b_1,…,b_n]$, then $Be_j=b_j$, we get \[\left.\begin{aligned}\det[Ba_1,…,Ba_n]&=\det[a_{11}b_1,a_{22}b_2,…,a_{nn}b_n]\\&+\det[a_{21}b_2,a_{12}b_1,…,a_{nn}b_n]\\&+⋯\\&+\det[a_{n1}b_n,a_{n-1,2}b_{n-1},…,a_{1n}b_1]\end{aligned}\right\}n!\text{ terms}\] Factoring out the constants \[\left.\begin{aligned}\det[Ba_1,…,Ba_n]&=a_{11}a_{22}⋯a_{nn}\det B\\&-a_{21}a_{12}⋯+a_{nn}\det B\\&±⋯\\&+(-1)^{\small\frac{n(n-1)}2}a_{n1}a_{n-1,2}⋯a_{1n}\det B\end{aligned}\right\}n!\text{ terms}\]Factoring out $\det B$ we get $\det BA=\det B\det A$.
